
# Subtree of Another Tree

**Objective:** Write a function that checks whether or not tree *t* is a subtree of tree *s*

#### Example1:
```
Input:
s:
     3
    / \
   4   5
  / \
 1   2

t:
   4 
  / \
 1   2

Output: true
```
#### Example2:
```
Input:
s:
     3
    / \
   4   5
  / \
 1   2
    /
   0

t:
   4 
  / \
 1   2

Output: false
```
